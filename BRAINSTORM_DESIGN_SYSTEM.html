4️⃣ HIBP Display : Avertissement Critique

Message clé: "Ce mot de passe a fuité" doit être impossible à ignorer, mais pas agressif.

Option 1: Alert Strip Seul (minimaliste)

⚠️ Ce mot de passe a déjà fuité. Trouvé dans plus de 5 millions de fuites connues. Changez-le immédiatement.

✅ Compact, direct, mobile-friendly. ✗ Peut être oublié si l'utilisateur scroll.

Option 2: Badge + Message (équilibre)

🚨 FUITÉ Trouvé dans 5M+ fuites

Ce mot de passe a été compromis et ne doit pas être utilisé. Changez-le partout où vous l'avez utilisé.

✅ Visible dans résultats, actionnable. ✗ Peut se perdre si beaucoup de contenu.

✅ Recommandation: Stratégie 2-Level

1. Alert Strip en haut du formulaire (après input) = avertissement immédiat

⚠️ Mot de passe fuité détecté. Ce mot a déjà été compromis. Utilisez un autre mot de passe.

2. Badge + Message dans les résultats = réaffirmation

🚨 COMPROMIS Apparaît dans 5+ millions de fuites

Action immédiate: Changez ce mot de passe partout où vous l'avez utilisé. Consultez Have I Been Pwned pour voir où il a fuité.

Logique: Double confirmation. Utilisateur ne peut pas ignorer. Fonctionne sur mobile. Cohérent avec design system.

5️⃣ Indicateurs de Confiance : Honnêteté Visible

Défi: Comment montrer qu'on est honnête, transparent, sécurisé? Les users lambda ne lisent pas le petit texte.

🔒

100% Local

👁️

Transparent

Code source ouvert, aucun piège

🔐

Privé par Défaut

HIBP avec k-anonymité seulement

⚡

Rapide & Efficace

Fonctionne sans connexion

Placement Stratégique

  • En haut de la page: "🔒 Zéro transmission de données. 100% local." — immédiatement rassurant
  • Sous le champ mot de passe: "Votre mot ne quitte jamais votre navigateur"
  • Près du bouton calcul: "Aucun enregistrement, aucune publicité"
  • Footer: Lien vers code source + licence MIT

Design Pattern: Icônes + texte court. Pas de longues explications. L'utilisateur lambda doit comprendre en 3 secondes qu'on est honnête.

6️⃣ Hiérarchie Typographique : Clarté d'Abord

Principe: Pas de forme sans fonction. Chaque taille, poids, couleur = signal sémantique.

🎯 Votre mot de passe: Faible
Temps de crack estimé (12× GPU):
47 ans
Problèmes détectés:
❌ Pas de symboles spéciaux
✅ Mélange de majuscules et minuscules
Dernière mise à jour: il y a 2 secondes

Règles Typographiques

  • H1 (2rem, bold, blanc): Verdict principal ("Votre mot de passe: Faible")
  • H2 (1.25rem, semi-bold, gris): Sections ("Temps calculé", "Problèmes")
  • Hero (2.5rem, gradient): Le NOMBRE (47 ans) — vedette absolue
  • Body (1rem, gris moyen): Explications, contexte
  • Caption (0.85rem, gris clair): Métadonnées ("Mis à jour", "Source")

Logique Mobile-First: Sur 320px, la taille diminue proportionnellement, mais la hiérarchie reste. H1 = 1.25rem, Hero = 1.75rem, etc.
