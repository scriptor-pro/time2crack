✅ Avantages
  • Couleur = intention immédiate
  • Scannable en 1 coup d'œil
  • Hiérarchie visuelle claire
  • Contraste WCAG AAA
  • Simple à implémenter
❌ Inconvénients
  • 60 couleurs peuvent surcharger
  • Demande CSS dynamique (classes)
  • Pas de groupement logique
  • Toutes lignes au même poids

✅ Avantages
  • Structure logique claire
  • Meilleur pour comparaison intra-groupe
  • Highlight du "meilleur" visible
  • Moins de couleurs à gérer
  • Facile de scanner par attaque
❌ Inconvénients
  • Plus de hauteur (headers)
  • Moins compact
  • Demande des headers colspan
  • Peut être visuel surchargé
Option C: Highlight "Best Attack" + Fade Others

Les lignes avec ⚡ sont les plus rapides pour chaque attaque.
Les autres s'estompent pour laisser ressortir les menaces réelles.

✅ Avantages
  • Attention focalisée sur le danger
  • Très visual (rouge + fade)
  • Montre clairement "meilleur"
  • Moins de couleurs
  • Animation possible (fade)
❌ Inconvénients
  • Les autres algos moins visibles
  • Peut être frustrante (fade)
  • Dépend de la détection JS
  • Pas de structure logique
