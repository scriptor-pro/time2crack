Méthodes d'Attaque

Explorez les 7 méthodes d'attaque modélisées par Time2Crack.

Force Brute

Teste systématiquement toutes les combinaisons possibles de caractères jusqu'à trouver le mot de passe. Exhaustif mais très coûteux en temps pour les mots de passe longs.

Dictionnaire

Teste des mots courants et des listes de mots de passe divulgués (HIBP). Très efficace sur les mots de passe faibles basés sur des mots réels.

Hybrid

Combine l'attaque par dictionnaire avec des mutations (ajout de chiffres, symboles, variations). Capture les variations simples des mots courants.

Mask

Cible les structures prédictibles (ex: majuscule + minuscules + chiffres à la fin). Exploite les patterns humains courants de création de mots de passe.

Markov

Modèle probabiliste basé sur les chaînes de Markov. Apprend les séquences de caractères courantes dans les mots de passe réels pour les tester en priorité.

PCFG

Probabilistic Context-Free Grammar. Analyse les structures grammaticales des mots de passe humains pour générer les variantes les plus probables en premier.

Combinator

Concatène des mots du dictionnaire pour créer des passphrases. Efficace sur les mots de passe multi-mots comme "correct-horse-battery-staple".
