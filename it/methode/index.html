Metodi di attacco

Esplora i 7 metodi di attacco modellati da Time2Crack.

Forza lorda

Testare sempre tutte le possibili combinazioni di caratteri fino a trovare la password. Esauriente ma molto costoso nel tempo per lunghe password.

Dizionario

Test parole comuni e liste di password divulgate (HIBP). Molto efficace su password deboli in base a parole reali.

ibrido

Combina l'attacco del dizionario con le mutazioni (addizione di numeri, simboli, variazioni).

Maschera

Obiettivi strutture prevedibili (ad esempio maiuscolo + minuscolo + cifre alla fine).

Markov

Modello probabilistico basato su stringhe Markov. Impara sequenze di caratteri comuni in password reali per testarli prima.

PCF

Analizza le strutture grammaticali delle password umane per generare prima le varianti più probabili.

Combinatore

Concate parole del dizionario per creare passphrases. Efficace su password multi-parole come "corse-corse-battery-staple".
