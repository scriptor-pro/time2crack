Prototype GEN-C — Modal / Bottom-sheet

Un bouton CTA secondaire ("Générer un mot de passe") apparaît sous le champ de saisie. Sur mobile, il ouvre un bottom-sheet qui glisse depuis le bas ; sur desktop (≥ 540px), un dialog centré. Le modal contient les options complètes + une prévisualisation en temps réel. À la confirmation, le modal se ferme et le mot de passe s'insère dans le champ.
